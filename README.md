**Bootstrap Stopwatch** is a simple, clean, and fully functional web-based stopwatch application. It is built using standard HTML and vanilla JavaScript, with all visual components styled entirely by the **Bootstrap 5** framework.

This project is an excellent example of combining basic JavaScript time manipulation with modern, responsive CSS styling provided by a popular front-end framework.

---

## üöÄ Features

* **Precise Timing:** Tracks time in minutes, seconds, and milliseconds.
* **Standard Controls:** Includes **Start**, **Pause**, and **Reset** functionality.
* **Bootstrap 5 Styling:** Fully responsive and clean interface powered by the official Bootstrap CSS framework.
* **Vanilla JavaScript:** Lightweight implementation with no external JavaScript libraries (aside from the Bootstrap dependency itself).
* **Responsive Design:** Optimized for display on all screen sizes, from mobile devices to desktop monitors.

* **HTML5:** The core structure of the application.

## üèÉ Usage

The application is straightforward to use via the three primary buttons:

| Button | Action | Description |
| :--- | :--- | :--- |
| **Start** (Green) | Starts / Resumes | Begins timing from 00:00:000 or resumes from a paused time. |
| **Pause** (Blue) | Pauses | Temporarily stops the timer. The elapsed time is saved. |
| **Reset** (Red) | Stops & Clears | Clears the timer back to `00:00:000` and resets the controls. |

The time is displayed in the format: `MM:SS:mmm` (Minutes:Seconds:Milliseconds).
